ドットポスト｜ハッカソン作品

#Webアプリ

#デザイン

#ハッカソン

概要

2026年3月9日〜11日に、ECCコンピュータ専門学校で開催された「Web制作合宿2026」に参加しました。 この合宿では、複数の専門学校や大学の学生が集まり、チームでWeb制作に取り組みました。

制作テーマ

平成レトロ

平成初期〜中期に流行したカルチャーを再発明することをテーマに、 当時の文化を参考にしながら、現代の技術や表現を使って新しい形にアップデートするWeb制作に挑戦しました。

参加校

この合宿には、以下の学校の学生が参加しました。

  • • 東京電子専門学校（東京）
  • • ECCコンピュータ専門学校（大阪）
  • • 清風情報工科学院（大阪）
  • • トライデントコンピュータ専門学校（名古屋）
  • • 名古屋文理大学（名古屋）
  • • 岐阜協立大学（岐阜）
  • • 神戸電子専門学校（神戸）

異なる学校の学生と協力しながら制作を行うことで、普段の授業とは違う視点や考え方に触れることができました。

制作背景

課題

近年のデジタルコミュニケーションでは、 SNSやメッセージアプリの普及によって 手間をかけて作る体験や、温かみのある交流が少なくなっていると感じました。 そこで以下の課題を設定しました。

  • ① デジタルでは「作って遊ぶ体験」が少ない
  • → 自分で作る楽しさを体験できるサービス
  • ② オンラインの交流がそっけない
  • → 時間をかけて作った作品やメッセージを届けられる仕組み

これらの課題を解決するため、 メッセージ付きアイロンビーズ作成アプリを企画しました。

企画内容

本作品は、平成レトロをテーマにしたメッセージ付きアイロンビーズ作成アプリです。 ユーザーはドットでビーズ作品を制作し、 メッセージと一緒に送ることができます。

ターゲット
  • • 10代後半〜20代後半
  • • 平成文化を知る親世代

平成文化の懐かしさと、 現代のデジタル表現を組み合わせたサービスを目指しました。

こだわったところ

ロゴデザイン

アプリ名は「ドットポスト」をベースに 「.ポスト」という表記にしました。 ロゴの「.」の部分をビーズのモチーフにすることで、 アプリのコンセプトであるドット表現を視覚的に伝えています。

配色

配色では、以下を意識しました。

  • • 純白や黒を避け、目に優しい色を使用
  • • カラフルなビーズ作品を邪魔しないデザイン
  • • 温かみや手作り感を感じる配色

制作画面はビーズの色が映えるよう 白をベースにしたシンプルなUIにしています。

  • MAIN COLOR

    #CEADA8

  • ACCENT COLOR

    #C47459

  • TEXT COLOR

    #AB675C

  • BASE COLOR

    #FEFEFE

テンプレートデザイン

ユーザーが簡単に作品を作れるようビーズテンプレートを用意し、初心者でも制作しやすくするとともにデザインの幅を広げられるようにしました。

今後の展望

今後の展望として、レスポンシブ対応や新規登録・フレンド機能の追加、データベースの構築などの機能拡張を検討しています。また、作業中に楽しめるBGM機能や送信完了画面でのアニメーションを実装することで、より快適で楽しいユーザー体験を提供できるアプリへと発展させていきたいと考えています。

感想

初めてのグループ制作だったため、コミュニケーション面も含め大変なことが多くありました。また、他校の学生のレベルの高さを感じ、自分にとって大きな刺激になりました。その中でも自分の強みを活かしながら制作に取り組みました。さらに、自分の知らなかったツールや制作方法についても知ることができ、学びの多い3日間となりました。
